# Results Directory

This directory is reserved for local experiment outputs.

Expected run artifacts may include:

- `results/<run_name>/config.yaml`
- `results/<run_name>/metrics.json`
- `results/<run_name>/logs/`
- `results/<run_name>/checkpoints/`

Only `results/.gitkeep` and this README are intended to be tracked in git.
Generated run artifacts, logs, checkpoints, and derived outputs should not be committed.
